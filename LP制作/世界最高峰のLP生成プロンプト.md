以下の条件で、コンバージョン率を最大化する最高品質のランディングページを作成してください：

【基本情報】
- 商品/サービス名：[具体的な名称]
- ターゲット顧客：[年齢層・職業・年収・課題]
- 価格帯：[金額とプラン構成]
- 主要ベネフィット：[顧客が得られる最大の価値3つ]
- 競合との差別化要素：[独自の強み]

【デザイン要件】
- デザインテイスト：[ラグジュアリー/モダン/信頼重視/先進的/親しみやすい など]
- カラーパレット：[メインカラー、アクセントカラー、または業界標準]
- 参考サイト：[URLまたは具体的なブランド名]

【構成要素】必須セクション
1. ファーストビュー：キャッチコピー（10文字以内）+ サブコピー（30文字以内）+ 強力なCTA
2. 課題提起：ターゲットの痛みを3つ箇条書き
3. ソリューション提示：商品/サービスがどう解決するか
4. ベネフィット詳細：数値・根拠付きで3-5項目
5. 社会的証明：実績数値・導入企業ロゴ・お客様の声3件
6. 特徴/機能：アイコン付きで6項目
7. 料金プラン：2-3プランを比較表で
8. よくある質問：5-7項目
9. 最終CTA：限定性・緊急性を含む強力な訴求

【コピーライティング原則】
- PASONAの法則またはAIDCAモデルを適用
- 具体的な数値とデータを多用
- 感情に訴えつつ論理的な裏付けも提示
- ベネフィットは「特徴」ではなく「顧客の変化」で表現
- CTAは行動を明確に指示（例：「今すぐ無料体験を始める」）

【技術要件】
- レスポンシブ対応（モバイルファースト）
- セクション間のスムーズなスクロールアニメーション
- CTAボタンはホバー時にマイクロインタラクション
- 読み込み速度最適化（画像はWebP、遅延読み込み）
- アクセシビリティ対応（WCAG 2.1 AA準拠）

【心理トリガー】必ず組み込む
- 希少性：「先着50名限定」
- 緊急性：「72時間限定オファー」
- 社会的証明：「導入企業3,000社突破」
- 権威性：「業界最高峰の専門家が開発」
- 返報性：「今なら無料特典3つプレゼント」

【測定・最適化】
- 各CTAボタンに計測タグ設置箇所を指定
- ヒートマップ分析用の重要セクションをマーク
- A/Bテスト候補要素を2箇所提案

【成果物】
- React + Tailwind CSS の単一ファイル
- 全セクションの完全実装（ダミーデータ含む）
- プロフェッショナルな画像プレースホルダー
- すぐにデプロイ可能な状態

【品質基準】
- デザイン：Apple、Stripe、Airbnbレベルの洗練度
- コピー：有名コピーライターレベルの訴求力
- UX：直感的で迷わない導線設計
- 技術：エラーゼロ、パフォーマンススコア95+
